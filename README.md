# pocket-swing

A small mobile-first web app.

## Develop

```bash
npm install
npm run dev
```

Open the printed local URL. The app uses the device camera, so grant camera
permission when prompted.

> Camera access requires a secure context. `localhost` works on desktop; to test
> on a phone over your network you'll need HTTPS (use a tunnel or an HTTPS dev cert).

## Build

```bash
npm run build
npm run preview
```
